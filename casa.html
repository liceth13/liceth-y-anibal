             VINTON CERF
               (Biografía)

Científico Estadounidense, nació en Connecticut en 1943. Se graduó en Matemáticas por la Universidad de Stanford en 1965. Después de trabajar durante un corto periodo en IBM, entró en el departamento de computación de la Universidad de California Los Ángeles para realizar un Master Científico y su Tesis Doctoral.

Trabajó en el motor de cohete F-1 que sirvió para propulsar el cohete Saturno V que viajó a la luna, Cerf también es el creador del protocolo TCP/IP.

Descubre el universo del Software

El software es el conjunto de programas, datos y procedimientos que permiten el funcionamiento lógico de una PC.
El software es el soporte lógico de un sistema informático porque comprende el conjunto de los componentes lógicos necesarios que hacen posible la realización de una tarea.  

Existen dos tipos de software: Software de aplicación y  Software de operación

Software de aplicación:
Está relacionado con los programas que utilizan los usuarios del sistema para realizar sus trabajos o tareas. Entre los diferentes
programas de aplicación, que encontramos en el mercado se encuentran los manejadores de bases de datos, los procesadores de texto, los programas de diseño gráfico, las hojas de cálculo, las aplicaciones administrativas de empresa y una variedad de utilidades que facilitan y hacen más eficiente el trabajo hoy en día. 

 Ejemplos de Software de aplicación:

Microsoft Word ​(​Procesador de texto)
Microsoft Excel ​(​Hoja de cálculo​)
Microsoft Access ​(​Manejador de bases de datos​)
Microsoft PowerPoint ​(​Diseño de presentaciones​)
Paint  ​(​Diseño gráfico​)
Oracle ​(​Manejador de bases de datos​)
Visual Basic ​(​​​Lenguaje de programación​)
Internet Explore​(​Navegador de internet​)

Software de operación:
Está relacionado con el Sistema Operativo de la máquina. Es decir el software de operación hace que los dispositivos físicos del ordenador, tales como la pantalla, el teclado, el mouse y el resto de elementos funcionen. Este facilita a los usuarios el uso de la computadora, sirviendo de interfaz o intermediario.

El desarrollo del software implica un proceso denominado ciclo de vida de desarrollo de software, que los gestores de proyectos utilizan para delinear las tareas y garantizar que el software satisfaga las necesidades de los usuarios.

El hardware   

El hardware es el componente físico y tangible de una computadora. Está constituido por el conjunto de circuitos electrónicos y componentes electromecánicos de una PC.


Tipos de hardware:

 Hardware de procesamiento: Es aquel que se encarga de ejecutar las instrucciones que se den desde el propio software del dispositivo. En otras palabras, la misión del hardware de procesamiento es realizar las ordenes que se vayan dando por parte del usuario del dispositivo vía software. Este tipo de unidades físicas son claves para el funcionamiento de un dispositivo, ya que son considerados el cerebro del dispositivo.

De forma técnica, el hardware de procesamiento suele ser la CPU del dispositivo, ya que es esta quien procesa los movimientos del dispositivo.
También se dan otro tipo de hardware de procesamiento como lo son las propias tarjetas dedicadas, ya sean de sonido, gráfica, etc.

De las funciones que lleva a cabo el hardware de procesamiento:
-Ejecución de todas las instrucciones del usuario.
-Administración de programas en segundo plano.
-Interconectar las distintas partes del hardware.

- Hardware de almacenamiento : Son los dispositivos físicos que permiten guardar información de manera permanente o temporal. Estos dispositivos permiten el acceso rápido a los datos almacenados en ellos y proporcionan una forma conveniente de guardar información que se puede utilizar más tarde.

Principales tipos de hardware de almacenamiento:

-Discos duros (HDD o SSD)
-Memorias USB portátiles
-Tarjetas de memoria SD
-Unidades de estado sólido SSD

Hardware de entrada: El hardware de entrada se refiere a los dispositivos que permiten al usuario introducir datos e instrucciones en un sistema informático. Ejemplos comunes de hardware de entrada incluyen:

-Teclado
-Ratón
-Escáner
-Micrófono
-Cámara web
-Pantallas táctiles
-Lectores de código de barras

Redes

Las redes, en el contexto de la tecnología, son sistemas de comunicación que conectan a distintos dispositivos entre sí para compartir información, recursos y servicios. Esto puede ser a través de internet, redes locales (LAN), redes inalámbricas (Wi-Fi), entre otras. Las redes permiten la comunicación y el intercambio de datos entre computadoras, teléfonos móviles, tabletas y otros dispositivos electrónicos.

Algunos de los tipos de redes que podemos encontrar son:

1. Redes LAN (Local Area Network): Son redes que conectan dispositivos en un área geográfica limitada, como una oficina, escuela o hogar.

2. Redes WAN (Wide Area Network): Estas redes abarcan áreas geográficas más extensas, como ciudades, países o incluso a nivel global. Internet es un ejemplo de una WAN.

3. Redes inalámbricas (Wi-Fi): Permiten la conexión de dispositivos de forma inalámbrica, utilizando tecnologías como Wi-Fi o Bluetooth.

4. Redes de área personal (PAN): Son redes de corto alcance que conectan dispositivos personales, como teléfonos móviles, computadoras portátiles y accesorios Bluetooth.
